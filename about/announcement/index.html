    Project Announcement¶

    Introduction¶

    On August 3, 2010, Garrett D'Amore announced the illumos project, "a community effort of some core Solaris engineers to create a truly open source Solaris by swapping closed source bits of OpenSolaris with open implementations."

    In January, 2012, he reflected on Experiences Starting an Open Source Operating System in a talk at SCALE 10x.
